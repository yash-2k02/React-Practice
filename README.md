# Conditional Rendering in React

This branch demonstrates different methods of performing **conditional rendering** in React.

## Concepts Covered

- **if-else rendering**: Using standard JavaScript `if-else` to conditionally render components.
- **Ternary operator rendering**: Conditionally rendering JSX using the ternary (`? :`) operator.
- **Logical AND (`&&`) rendering**: Rendering elements only when a specific condition is true.
