# Props Trailing in React

This branch demonstrates how to pass data using **props** from a parent component to a child component in React.

## Concepts Covered

- **Passing data via props**: Sending data from the parent (`App.jsx`) to the child (`Card.jsx`) component.
- **Rendering dynamic content**: Displaying content in the child component based on the received props.

## Implementation Details

- A **data array** is defined in **App.jsx** containing information of events (title, description, date).
- The data is passed as props to the **Card** component.
- The **Card** component receives and uses the props to render dynamic UI on the page.
