# Working with useState in React (Form Handling)

This branch demonstrates how to use the **`useState` hook** in React to manage form data.

## Concepts Covered

- **useState hook**: Handling form input values using React state.
- **Form handling in React**: Capturing user input and updating state variables in real-time.
- **Form submission**: Collecting all form data and logging it as an object upon form submission.

## Implementation Details

- A **Form component** contains input fields for capturing user data.
- Each input field is controlled using its own **`useState` hook**.
- On **form submission**, the values from each state variable are gathered into an **object format**.
- The object containing the form data is then **logged to the console** for inspection.
